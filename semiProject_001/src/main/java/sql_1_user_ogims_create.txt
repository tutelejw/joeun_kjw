-- 0. 사용자 삭제 초기화 (system/system 접속해서 진행)
DROP USER ogims CASCADE;


-- 1. 사용자 생성 (system/system 접속해서 진행)
CREATE USER ogims IDENTIFIED BY "ogims";

-- 2. 기본 롤 부여 (CONNECT: 로그인용, RESOURCE: 테이블/시퀀스 등 객체 생성용)
GRANT CONNECT, RESOURCE TO ogims;

-- 3. 추가 권한 부여 (뷰, 프로시저, 트리거 등)
GRANT CREATE VIEW TO ogims;
GRANT CREATE PROCEDURE TO ogims;
GRANT CREATE TRIGGER TO ogims;
