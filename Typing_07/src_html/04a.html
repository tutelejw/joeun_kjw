HTML : Hypertext Markup Language

- Hyper+Text (활동적, 이동성+문자)
   : 문자를 동적으로 연결 , 사용자의 선택에 따라 특정한 정보, 관련된 부분으로 이동
- Markup Language
   : 조직화 된 문서 , 표준화된 문서













Tag

   : HTML 페이지에서 Element 객체(?)를 만드는데 사용
   : 사용되는 Tag 는 W3C(World Wide Web) Consortium 표준 제정.













etc....

   : Attribute : Tag 에 추가정보를 부여.
   : Element : Tag , Tag 추가정보를 기술한 Attribute를 이용 만들어진 객체(?)
